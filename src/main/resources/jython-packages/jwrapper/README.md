jwrapper
===============================

version number: 0.0.1
author: jwrapper

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install jwrapper


Or clone the repo:

    $ git clone https://github.com/yishenggudou/jwrapper.git
    $ python setup.py install
